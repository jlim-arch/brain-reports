# brain-reports

Formatted research/report pages generated by the capture brain (VPS) from Hilite highlights.
Faithful static port of ad-dashboard-fluid's brief look (see styles.css — tokens lifted from
ad-dashboard-fluid/app/globals.css + components/brief.tsx). Zero build: static HTML + CSS.

- `styles.css` — the design system (single source of truth for all reports)
- `report-template.html` — skeleton the brain fills → `reports/<slug>.html`
- `index.html` — auto-updated list of reports

Deploy: connected to Vercel git integration. The brain pushes to this repo (repo-scoped
deploy key) and Vercel auto-builds. No Vercel token lives on the VPS.
